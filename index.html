Cursos

  • JavaScript na Web: armazenando dados no navegador

    Alura:  

    Período: 17/05/2023 a 15/07/2023

    No curso JavaScript na Web: Armazenando dados no navegador, desenvolvi a aplicação Mochila de Viagem, onde aprendi a salvar e recuperar informações diretamente do navegador usando as APIs localStorage e sessionStorage.   • JavaScript: criando requisições

    Alura:  

    Período: 04/10/2023 a 02/04/2024

    No curso JavaScript: Criando Requisições, aprendi a transformar páginas estáticas em interfaces dinâmicas utilizando requisições HTTP. Trabalhei com o json-server para simular uma API e implementei operações GET e POST para consumir e enviar dados. Aprimorei o uso de JavaScript assíncrono, manipulação do DOM, tratamento de erros e criação de funcionalidades como cadastro de vídeos e busca dinâmica.       Habilidades Adiquiridas

    • Manipulação de APIs com JavaScript
    • Trabalho com JavaScript assíncrono
    • Integração entre front-end e back-end simulado
  • React: como os componentes funcionam

    Alura:  

    Período: 17/09/2024 a 03/10/2024

    No curso React: como os componentes funcionam, aprofundei meu entendimento sobre a estrutura e o funcionamento interno do React.
